Preface

Back to contents

The Ian Foster Services Work Manager is a bespoke application designed to make the process of managing a plumbing business easier. It provides features such as customer and job databases, invoice and receipt generation with automatic emailing, and monthly report generation.

This user guide will firstly explain how to install the program on your operating system. Initial set up and settings will then be explained, followed by a guide of typical use of the program. Towards the end of the guide, troubleshooting is discussed.

It is strongly recommended that you read this entire guide before using the program, so that you can make best use of the program.

Installation

Back to contents

Installing the program on your computer is simple. Please make sure that your computer meets the following minimum system requirements before continuing.

Minimum System Requirements
  • 1Ghz single core processor or equivalent
  • 512MB of RAM
  • 80MB of free hard drive space
  • Graphics card and monitor capable of a 1024x768 screen resolution
  • A 64-bit Ubuntu (version 11.10 or greater recommended) or Windows (Vista or 7 recommended) installation
NOTE: The program may work on other operating systems, but only those mentioned are officially supported.

Back to Installation

Installing on Ubuntu

Step 1

You will have been provided with the ZIP file 'IanFosterServices_WorkManager.zip'. Open this file. You will see two folders: 'ubuntu' and 'windows'.

Step 2

Drag the 'ubuntu' folder into a location of your choice (the Desktop was used in the provided image). Open the folder and double click the file 'install.sh'. A dialog will appear, asking whether you want to run the file or display its contents. Click 'Run'. A dialog asking for your password should appear. Enter your password when this happens.

Step 3

The program should have installed successfully. Open the Ubuntu Dash (or the applications menu if you are not using Gnome, KDE, etc. instead of the Unity interface). You should be able to find the application if you navigate to the 'Installed' applications section, or start to type in 'Ian Foster Services Work Manager' in the 'Search Applications' input field. If you are not using the Unity interface, then the program will be able to be found in the 'Office' category of your applications menu.

Step 4

Once you have found the program, you can run it just like any other application installed on your computer. Do the initial set up and you will be able to use the program (for more information on the initial set up, go to the First Time Run section). You may now delete 'IanFosterServices_WorkManager.zip' and the 'ubuntu' folder.

Back to Installation

Installing on Windows

Step 1

You will have been provided with the ZIP file 'IanFosterServices_WorkManager.zip'. Open this file. You will see two folders: 'ubuntu' and 'windows'.

Step 2

Drag the 'windows' folder into a location of your choice (the Desktop was used in the provided image) and open it. Firstly, you will need to run either the 'vcredist_x64' installer if you are using a 64-bit Windows installation or the 'vcredist_x86' installer if you are using a 32-bit Windows installation. This must be installed in order for the program to be able to send emails correctly. If the installer gives the option to either 'Repair' or 'Remove', choose the 'Repair' option (on most systems this will not occur, so do not worry if you are not given this option). If the installer asks to restart your computer, let it restart.

Step 3

Once you have installed either 'vcredist_x64' or 'vcredist_x86', run the 'IanFosterServices_WorkManger_Installer' file in the 'windows' folder. You will then be guided through the process of installing the program on your computer.

Step 4

Verify that the program has been installed correctly by checking that there is an 'Ian Foster Services Work Manager' icon on your Desktop and also an 'Ian Foster Services Work Manager' listing in the Start Menu.

Step 5

You can run the program just like any other application installed on your computer. Do the initial set up and you will be able to use the program (for more information on the initial set up, go to the First Time Run section). You may now delete 'IanFosterServices_WorkManager.zip' and the 'windows' folder.

Settings

Back to contents

This is the settings screen. You will have access to this screen when the program is run for the first time, and can be accessed from the home screen at any time. For more information about the home screen, go to the Home Screen section.

Directories

The directory settings allow you to specify the folder where you would like files generated by the program to be saved. It is recommended that you choose a folder that you can remember and have easy access to. If you change a directory setting, the files in the old folder will automatically be moved to the new location for you.

Database

The database directory is the folder that contains the main database for the program. This folder should be secure, as it is the location where all of your customer and job records will be stored.

Backup

The backup directory is the folder that contains all of your backups. It is recommended that this folder is on an external device, such as an external hard disk, or is synchronised with an online cloud service. For more information about backups, go to the Backups section.

PDF

The PDF directory is the folder that contains all of the PDF files generated by the program; the invoices, receipts and reports. This folder should be in a location that is easily accessible, such as your 'Documents' or 'My Documents' folder, because you will likely want to view these PDFs at a later date, or print them out so that hard copies can be filed or sent to customers.

Back to Settings

Password

By using the 'Change Password' form, you can set a password that will be required by the program on start up in order for the user to gain access. This is useful if the program is installed on a shared computer, and you would like to keep the data safe from other users.

Passwords must be at least four characters long. It is recommended that you use at least eight characters and that you use capital letters, numbers and special characters in order to make your password more difficult to guess.

Back to Settings

Email

A useful feature of this program is that it allows you to send emails to customers. You can send invoices, receipts and even reminders if they have not paid for your work in a while. The email settings allow you to customise the content of these emails and to specify which email account to send the emails from.

Email Content

The program allows you to customise the subject and body of the invoice, receipt and reminder emails. To choose which email content you would like to modify, select the relevant tab to the left.

Email Host

The Email Host settings must be filled in correctly in order for the program to send emails.

The 'Username' field requires your email account username (this is typically the first part of your email address, before the '@' sign).

The 'Password' field requires the password for your email account.

The 'Host address' field requires the address of the outgoing server for emails (it usually looks like 'smtp.myemailhost.com', or similar).

The 'Host port' field requires the port number for SSL connections to the outgoing email server (this is usually 465, which the program defaults to). If you do not know the details for the outgoing server of your email host, please visit the website of the host (which is usually the second part of your email address, after the '@' sign).

Using The Program

Home Screen

1 Menu
The menu consists of three sections. The Management section allows you to visit different places in the program to manage jobs, customers, expenses and VAT history. The Reports section allows you to generate monthly financial reports and also provides the Report Wizard, which allows you to generate a report for a specific time period. The Options section allows you to visit the settings screen, access this help document, and exit the program.

2 Calendar
By selecting a date on the calendar, you can change the period of time that the program manages data on. Changing the month displayed on the calendar will change the displayed financial totals. Reports are generated with respect to the selected month, and only jobs and expenses in the selected month will be displayed on their index screens. Changing the day causes the jobs and tasks listed at the bottom of the screen to change.

3 Jobs and Tasks
These boxes will display a list of all of the jobs and tasks that need to be done for a particular day, specified by the calendar. You can double click any of the jobs or tasks displayed here to see information about them.

4 Financial Totals
This section gives a quick overview of the financial totals for a particular month, specified by the calendar. For a more detailed report for the month, use the 'Generate report' option on the menu.

5 Reminder Notice
This notice will appear if there are customers that have not paid for the work that you have done for them after seven days. This will send an email reminder to the customers, with the subject and message body specified in the settings.

Managing Customers

To view the customer index screen, click on 'Customers' in the home screen menu. You will be given a list of all of the customers that you have records of, in the alphabetical of surnames. From this screen, you can view the details of a specific customer by double clicking a name and add new customers to the database by clicking 'Add new customer' at the bottom of the screen.

Viewing Customers

To view the details for a specific customer, double click their name in the customer index. The customer details screen will then be displayed. To the left is the customer's personal information, and on the right is a list of all of the jobs that have been done or need to be done for the customer, sorted by date, descending. You can edit the customer's details by clicking the 'Edit' button, or delete the customer by clicking the 'Delete' button. When a customer is deleted, all of the jobs associated with that customer will be deleted as well. To view a job for the customer, double click the job in the job list. To add a new job for the customer, click the 'Add job' button. For more information about viewing and adding jobs, go to the Managing Jobs section.

Adding a New Customer

To add a new customer to the database, click on the 'Add new customer' button on the customer index screen. The customer form will then be displayed. Fill in as many fields as possible before clicking the 'Submit' button to save the new customer. The 'Address Line 2' and 'Mobile Phone' fields are optional. Any fields that do not have valid inputs will be highlighted in red. Hover the cursor over the field to find out how you can correct the problem. Here is an example:

Editing a Customer

To edit the details of a customer, go to the customer's details screen, and then click on the 'Edit' button. The customer form will then be displayed, with the customer's details already filled in. Change the inputs of the fields as required, and then click the 'Submit' button to save your changes. The 'Address Line 2' and 'Mobile Phone' fields are optional. Any fields that do not have valid inputs will be highlighted in red. To find out what is wrong with a particular input value, hover the cursor over the field and a tool tip will be displayed.

Deleting a Customer

To remove a customer from the database, go to the customer's details screen, and then click on the 'Delete' button. You will then be asked whether you are sure that you want to delete the customer. Only click 'Yes' if you are completely sure. When a customer is deleted, all of the jobs associated with that customer will be deleted as well. They will not be able to be retrieved.

Managing Jobs

To view a list of all of the jobs for the month selected on the home screen calendar, click on 'Jobs' in the home screen menu. A list of jobs will be displayed, sorted by date, descending. To view the details of a particular job, double click its entry in the list, and to add a new job, click the 'Add new job' button.

Viewing Jobs

To view the details of a particular job, double click its entry in the job index. The details of the job will then be displayed. To view the details of the customer that the job is being done for, click the 'View customer details' button. To view details of a part or task associated with the job, double click an entry in their respective lists (for more information about parts, go to the Managing Parts section, and for more information about tasks, go to the Managing Tasks section).

Once you have completed a job, you can indicate this by clicking the 'Mark as done' button. This will then give the job the completion state of 'Done - not paid'. It will then give you the option to generate an invoice, which can be emailed to the customer by clicking the 'Send invoice' button, or viewed in your PDF reader by clicking the 'View invoice' button (for more information about invoices, go to the Invoices section).

Once the job has been paid for, click the 'Mark as paid' button. The following dialog will be displayed, allowing you to set the payment date and payment method:

The job will be given the completion state of 'Done - paid' and the payment date and payment method will be recorded. It will then give you the option to generate a receipt, which can be emailed to the customer by clicking the 'Send receipt' button, or viewed in your PDF reader by clicking the 'View receipt' button (for more information about receipts, go to the Receipts section).

Adding a New Job

To add a new job to the database, click the 'Add new job' button on the job index screen. An empty job form will then be displayed. Fill in as many fields as possible. Any fields that do not have valid inputs will be highlighted in red. To find out what is wrong with a particular input value, hover the cursor over the field and a tool tip will be displayed.

A drop down box listing all of the customers is given towards the top of the form. To view the details of the selected customer, click the 'View customer' button. To add a new customer to the database, click the 'Add new customer' button. After saving the new customer to the database, the customer's name will be added to the list.

To add a part or task to the job, click the 'Add part' and 'Add task' buttons respectively (for more information about parts, go to the Managing Parts section, and for more information about tasks, go to the Managing Tasks section).

The total charge excluding VAT is calculated from the labour charge and the total cost of the parts excluding VAT. The VAT is taken from the VAT rate history (for more information about the VAT rate history, go to the Managing VAT Rates section). The total charge including VAT is the sum of the labour charge, the VAT of the labour charge calculated from the VAT rate, and the cost of the parts including VAT.

To save the new job to the database, click the 'Submit' button.

Backups

Back to contents

Keeping backups of your data is extremely important. Thankfully, the program takes care of backups for you, so you do not need to worry about making backups yourself. Is is recommended that you have the program settings set so that backups are made to an external devise or to a folder that is synchronised with an online cloud service.

At the beginning of every week, the program creates a new folder in the backup directory (specified in the settings) with the year and week number. Every time the program is closed, it copies files in the database folder (specified in the settings) and puts them in the current week's backup folder. This assures that the latest data is backed up, while at the same time creating a backup history.

If your database files are corrupted, deleted, or lost in any other way, then copy the backup files for the latest week into the folder that the program is stored in. You will then be able to continue using the program with minimal data loss.

An error message 'Could not open file something.dat' appears. How do I fix this?

Either the program could not find the file or the file being used by another program. First, make sure that the file is present in the database folder that is specified in the settings. If the file is present, then make sure that the file is not being used by another program. If the file is not present in the database folder, then either copy the latest version of the file from the backups folder into the database folder, or restart the program (the program will automatically recreate the missing database file for you on start up).

Back to Troubleshooting and Frequently Asked Questions

When sending an email, the 'Sending emails' dialog box pops up and doesn't go away after waiting a long time. How do I fix this?

There is likely to be a problem with your email host settings, or with the customer's email address that the email is being set to. To check your email host settings, select the 'Email Host' tab on the settings screen. If your email host settings are correct, check the email address of the customer. Once you have verified that these are correct, close the program, making sure to cancel the sending of the emails. Then reopen the program and send the emails again.

Back to Troubleshooting and Frequently Asked Questions

For specific advice on generating and viewing invoices or receipts, go to the Invoices or Receipts sections, respectively. For advice on generating and viewing reports, go to the Reports section.

How can I synchronise the program across multiple computers?

To have the database, backups, settings and/or PDFs synchronised across multiple computers or operating systems, you will need to specify a folder that is automatically synchronised across multiple computers in the settings for database/backup/PDF directories (for more information about folder settings, go to the Directory Settings section). On all of the computers that the program is installed on, the settings will need to point to this folder. It is recommended that you set up all of your computers for this before using the program fully, so that there is no risk of data accidentally being overwritten. If you have the database stored in a synchronised folder, make sure that when setting up a new installation of the program on another computer, you choose 'Delete file and keep existing file' if a 'File Already Exists' dialog appears when setting the directory settings to point to the synchronised folder.

A recommended service for synchronising files across multiple computers is Dropbox. Please visit www.dropbox.com for more information.
